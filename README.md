# 🏦 Simple Banking Terminal

This is a small C# project I built to practice real-world programming logic. Instead of just doing basic math, I wanted to see if I could simulate the core flow of a banking terminal.

### What it does:
* **Security:** It generates a random One-Time Password (OTP) that the user must confirm to access their data.
* **Projections:** It takes a deposit and calculates how much it would grow over 10 years at a 5% interest rate.
* **Formatting:** It automatically creates a unique Account ID by "clipping" parts of the user's name and balance.

### Why I built this:
I used this project to get comfortable with converting user input, handling string methods like `.Substring()`, and using `Math.Pow` for financial formulas. It's a simple start, but it helped me understand how different layers of logic (security, math, and text) work together in a single program.
